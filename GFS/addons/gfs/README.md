# File Structure Addon

## Introduction
Simple godot addon to create main folders in a godot project.
	
## Technologies

- Godot 4.2.

## Features
Will create these folders:
- Assets.
- Assets/Audio.
- Assets/Fonts.
- Assets/Art.
- Scenes.
- Scenes/Levels.
- Scenes/UI.
- Scenes/Objects.
- Scenes/Characters.
- Scenes/Enemies
- Scripts.
## Setup

- Just put the addon in addons folder.

## Usage

- Just enable the plugin.
